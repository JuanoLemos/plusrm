# Architecture Decision Records

Índice de decisiones de arquitectura del proyecto.

| # | Título | Status | Fecha |
|---|--------|--------|-------|
| 001 | [Título] | Accepted | YYYY-MM-DD |

## Estados

- **Proposed** — Propuesta, en discusión
- **Accepted** — Aceptada, en vigor
- **Deprecated** — Obsoleta (reemplazada por otra ADR)
- **Superseded** — Reemplazada por ADR-[NNN]

## Template

Para crear una nueva ADR, copiar `adr-template.md` y seguir el formato.
